## Definisi

Single-page Application (SPA) adalah jenis web yang berinteraksi dengan pengguna secara dinamis menulis ulang halaman web saat ini dengan data baru dari server web, ketimbang memuat seluruh halaman baru.

## Contoh

1. Pengguna mengunjungi website
2. Halaman web muncul
3. Pengguna klik tombol dengan tujuan pindah halaman
4. Halaman yang dituju muncul dengan data dan tampilan berbeda dan browser tidak reload

Tapi sebenarnya halaman tersebut tidak benar-benar pindah. Hanya saja ditulis ulang komponen-komponen yang berubah bersamaan dengan data baru.

Demo: https://enterprise.kredibel.co.id/

### Buktikan dengan inspect element.

## JSX

JSX adalah HTML-like syntax untuk menulis component React.

Tadinya begini:

```js
function component() {
    return React.createElement('h1', null, 'Hello World');
}
```

Dengan JSX:

```js

function component() {
    return <h1>Hello World</h1>;
}
```

## Definisi

Next JS adalah framework JavaScript berbasis React untuk membangun website SPA.

Fitur:
- Pre-rendering (SSR or SSG)
- API routes
- File-system routing
- Fast refresh
- Built-in CSS support
- Typescript support
- Image optimization
- i18n
- Code splitting
